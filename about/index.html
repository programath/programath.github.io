Tuopu Wen

I am interested in Hardware Design, Deep Learning. Use some mathematic tricks to help solving my problems.

Experience

  • Tsinghua University, 2014~

Project

FPGA-Oriented Low-Bit Quantization for Convolution Neural Network
A hardware-friendly quantization strategy. It is based on the caffe Ristretto framework. In Ristretto, many tasks can be quantized into 8bit without much degradtion by finetuning trick.
My strategy is to quantize the full precision network into 4bit codebook and replace multiplication by bit shift operation.

Architecture Design of Winograd Convolution Accelerator based on FPGA
It has been known that winograd algorithm can significantly reduce the number of multiplication compared to naive convolution implementation. We designed a winograd convolution framework
on zc706 by Vivado HLS tool chain, to run an online road lane detection task.(20fps).

Stock Trends Prediction Using Gaussian Process Regression
I try to used Gaussian Process Regression to modeling the behavior of stock’s market and predicting stocks’ prices.In this project, I proposed a method called piece-wise GPR to perform long-term prediction of stocks’ prices to fully utilize the periodicity of stock market.

The CNN Accelerator Design Based on FPGA
It was my first time to design a CNN accelerator running on FPGA. In this design, I transformed the convolution computation into matrix multiplication,
and used the parallelism of matrix multiplication to accelerate CNN. The accelerator finally got a 10x acceleration on computing performance.

Indoor Localization of Wheeled Robots and Drones
The project is designed for Tsinghua Electronic Design Competition. We managed to localizing wheeled cars using computer vision while UAVs using UWB technology with four anchors fixed. In this platform, we processed data from UWB, coded flight control and built the communication between the UAV and the server.
The all system can run in real-time and the accuracy of positioning reached within 5cm.

Modeling, Assessment and Prediction of Risks of Metabolic Syndrome
We modeled the metabolic reactions in human body. PDE, Matrix Completion, and SVM are used in modeling the reaction network. PDE was used to describe the behavior of reactions with the influence of catalyzer.
We also used Matrix completion to predict the missing information of samples. During the information prediction, L1 norm was added into the constrains to obtain the sparse result.

Disorganized Video Frames Rearrangement
The task is to resort the disorganized video frames. In this project, techniques like K-means and PCA were used to cluster different scenes, and optical flow trick was used to
determine the order of frames.

Air Hockey Robot Design of based on OpenCV
Designed a robot which could hit the fast moving ball(Air Hockey game). We controlled the robot arm to hit the ball. To find where the ball is, we fused the color and optical flow information.
The project finally won the second prize of Challenge Cup held in Tsinghua.
